## 媒体查询

根据某些条件(最常见的有查询 width 和查询是否为打印机),引入相应的 css 文件

1. css文件中

   `@media (min-width:100px) and (max-width: 300px){` 

   ​		body{   

   ​			background: white;  

   ​		}

   ​	`}`

2. 外链link

   ```
   <link rel="stylesheet" href="white.css" media="(min-width: 600px) and (maxwidth:400px)">
   ```
